## Description

A modern, interactive music player web application built with **HTML**, **CSS**, and **JavaScript**. This project features a sleek user interface with full music playback controls and playlist management capabilities.

## Features

- ✨ **Play/Pause Controls** - Start and pause your music with ease
- 🔊 **Volume Adjustment** - Control the audio output level
- 📋 **Playlist Management** - Manage and organize your music collection
- 🎨 **Modern UI Design** - Beautiful and responsive user interface
- ⌨️ **Keyboard Shortcuts** - Quick access controls for power users
- 📱 **Responsive Design** - Works seamlessly on desktop and mobile devices

## Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Advanced styling and animations
- **JavaScript (ES6+)** - Interactive functionality and DOM manipulation

## Getting Started

### Prerequisites

You only need a modern web browser to run this application:
- Chrome/Chromium
- Firefox
- Safari
- Edge

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Meshakdev/music-player-app.git
```

2. Navigate to the project directory:
```bash
cd music-player-app
```

3. Open `index.html` in your web browser

## Usage

1. Load your music files into the player
2. Click the play button to start playing
3. Use the volume slider to adjust sound level
4. Create and manage playlists
5. Enjoy your favorite music!
